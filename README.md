# BuildX – Prompt to App Generator

Turn plain English into fully functional web apps.

---

# About

BuildX is an AI-powered system that converts user prompts into interactive React applications with a live preview.

Built during DevClash Hackathon at Zeal College of Engineering and Research, Pune.

The goal was to simplify app development by allowing users to generate and modify apps using natural language.

---

## ⚡ Features

- 🧠 Generate apps from simple prompts  
- 🔄 Real-time app editing (modify UI via chat)  
- 🎨 Modern UI using Tailwind CSS  
- ⚡ Live preview without build tools  
- 🛠 AI refinement loop for improved UI/UX  
- 🔁 Smart fallback system for stability

  
---

## 🧱 Tech Stack

- Frontend: React (CDN-based)
- Styling: Tailwind CSS
- Backend: FastAPI
- AI: Groq API
- Language: JavaScript, Python

---

## 🏗️ How It Works

1. User enters a prompt  
2. AI generates React app code  
3. Code is validated and rendered in a live preview  
4. User can modify the app using follow-up prompts  
5. System updates the app in real-time
